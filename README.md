**PhysioPal**
PhysioPal brings professional physiotherapy guidance directly to your home through AI-powered exercise analysis. Using computer vision and pose estimation, it provides real-time feedback on posture, joint angles, and movement accuracy helping users perform exercises safely without needing in-person supervision.
Built with MediaPipe, OpenCV, and Streamlit, PhysioPal integrates Google’s Gemini API to deliver personalized exercise recommendations and adaptive guidance. Designed for accessibility, it requires only a standard webcam and no additional sensors or wearables.
Key Features:
• Real-time form correction and rep tracking
• Intelligent feedback based on joint-angle analysis
• Personalized exercise progression and recommendations
• Lightweight web interface for seamless user interaction
Tech Stack: Python, Streamlit, OpenCV, MediaPipe, Gemini API
